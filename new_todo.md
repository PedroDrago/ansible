### New firefox task:
1. remove firefox snap
sudo snap remove --purge firefox

2. remove firefox bin
sudo rm /bin/firefox

3.Add Mozilla Team PPA:

sudo add-apt-repository ppa:mozillateam/ppa

3. Install Firefox ESR:

sudo apt install firefox-esr
